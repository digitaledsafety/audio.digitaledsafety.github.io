# VCA Node

The VCA (Voltage-Controlled Amplifier) Node can be used to control the volume of an audio signal.

## Inputs

*   **Audio In**: The audio signal to be amplified.
*   **Gain CV**: A Control Voltage input to modulate the gain.

## Outputs

*   **Audio Out**: The amplified audio signal.

## Controls

*   **Gain**: Sets the base gain level. This value is added to any modulation from the Gain CV input.
