# Compressor Node

The Compressor Node can be used to compress the dynamic range of an audio signal.

## Inputs

*   **Audio In**: The audio input to the node.

## Outputs

*   **Audio Out**: The audio output of the node.

## Controls

*   **Threshold (dB)**: The threshold of the compressor in decibels.
*   **Ratio**: The ratio of the compressor.
*   **Attack (s)**: The attack time of the compressor in seconds.
*   **Release (s)**: The release time of the compressor in seconds.
