# LFO Node

The LFO Node can be used to create a low-frequency oscillator.

## Outputs

*   **CV Out**: The CV output of the node.

## Controls

*   **Frequency (Hz)**: The frequency of the LFO in Hertz.
*   **Waveform**: The waveform of the LFO. Can be one of `sine`, `square`, `sawtooth`, or `triangle`.
*   **Amount**: The amplitude of the LFO output.
